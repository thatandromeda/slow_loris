What's this?

This is a site that tells you all sorts of ways (not just committee appointments!) you can get involved with LITA, while feeding your craving for adorable baby animals.

What's LITA?

LITA is the Library & Information Technology Association, a division of the American Library Association.

What's the tech stack?

Written in Django, a web framework for Python. Deployed to Heroku. Code hosted on Github.

Who built it?

This site was built by Andromeda Yelton. Thanks go to Molly Tomlinson and Polly-Alida Farrington (for help finding cute animals) and Jason Griffey (for breaking it until it got better). Maybe you will help build it, too? You can submit new ways to get involved. Know how to improve the code? Pull requests welcome.

While I am active in LITA, this site is not affiliated with the association in any way.
